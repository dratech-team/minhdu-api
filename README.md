##  Cài đặt

1/ Cài nestjs cli
```bash
$ npm i -g @nestjs/cli
```

2/ Chạy docker compose để tải xuống môi trường mongodb
```bash
$ docker-compose up
```
3/ Cài node modules
```bash
$ npm install
```

## Running the app

```bash
# Run ở môi trường dev
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
